# Digimon Partner 🥚

Raise a Digimon partner that evolves as you code. Every save, edit, and terminal you open earns XP — watch your partner grow from a Fresh egg all the way to Mega.

![Digimon Partner](media/icon.png)

## How it works

Open the **Digimon Partner** panel from the activity bar (look for the Digivice icon on the left). Hatch your first egg and start coding.

| Action | XP |
|--------|----|
| Save a file | +5 XP |
| Edit code (every 5s) | +1 XP |
| Open a terminal | +3 XP |

Every **500 XP** you earn a new egg. Once your Digimon has enough XP it's ready to Digivolve — click it in the roster to choose its next form.

## Evolution

Your Digimon follows a full branching evolution tree across 6 stages:

- 🥚 **Fresh** → **In-Training** (20 XP)
- **In-Training** → **Rookie** (50 XP)
- **Rookie** → **Champion** (150 XP)
- **Champion** → **Ultimate** (400 XP)
- **Ultimate** → **Mega** (800 XP)

With **220 Digimon** available, every playthrough can be different.

## Features

- 🦕 220 Digimon with pixel art sprites
- 🌿 Branching evolution paths — same Rookie can lead to many different Champions
- 👁 Show/hide individual Digimon in the arena
- 🥚 Collect multiple Digimon — earn new eggs as you code
- ⚡ Evolution panel shows all available digivolution options with previews

## Debug Commands

Open the Command Palette (`Ctrl+Shift+P`) and search for **Digimon**:

- **Digimon: [Debug] Add Digimon** — instantly add any Digimon to your collection
- **Digimon: [Debug] Add XP** — add XP to all your Digimon
- **Digimon: [Debug] Validate** — run a data validation report
- **Digimon: Reset Partner** — reset everything and start fresh
